# 캘리브레이션 관련 — 멀티캠캘리브 사용방법

| 항목 | 내용 |
|------|------|
| 경로 | `wrappers/python/examples/box_dimensioner_multicam/` |
| 핵심 모듈 | `calibration_kabsch.py` |
| 데모 | `box_dimensioner_multicam_demo.py` |
| 역할 | 멀티캠캘리브 |

---

## 사전 조건

- Python 3.6+
- `pip install opencv-python numpy pyrealsense2`
- RealSense 카메라 **2대 이상**
- 공통 시야의 **체스보드** (데모 기본 6×9)

---

## 실행

```powershell
cd d:\study\librealsense\wrappers\python\examples\box_dimensioner_multicam
python box_dimensioner_multicam_demo.py
```

체스보드 크기가 다르면 스크립트 내 체스보드 파라미터를 수정한다.

---

## 워크플로

1. 모든 카메라 FOV에 캘리브 체스보드를 배치
2. 프로그램 시작 → Kabsch로 카메라 간 강체 변환(회전·이동) 추정 (`calibration_kabsch.py`)
3. 안내가 나오면 **체스보드 위에** 측정할 상자를 올림 (가로·세로는 보드보다 작아야 함)
4. 바운딩 박스의 길이·너비·높이(mm) 표시
5. 장치별 컬러 창에 박스 오버레이

---

## `calibration_kabsch.py` 역할

- 체스보드 3D↔카메라 좌표로 **Kabsch** 최적 회전·이동 계산
- 멀티캠을 하나의 월드 좌표계로 정렬하는 헬퍼 (단독 CLI가 아니라 데모에서 import)

참고: Kabsch W., 1976, Acta Crystallographica.
