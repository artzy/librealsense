# 캘리브레이션 관련 — 캘리브덤프 사용방법

| 항목 | 내용 |
|------|------|
| 경로 | `tools/enumerate-devices/` (`rs-enumerate-devices.exe`) |
| 역할 | 캘리브덤프 |
| 핵심 옵션 | `-c` / `--calib_data` |

---

## 사전 조건

- 빌드된 `build\Release\rs-enumerate-devices.exe`
- RealSense 장치 연결 (또는 `-p`로 bag 파일)

---

## 캘리브 덤프 실행

```powershell
cd d:\study\librealsense\build\Release
.\rs-enumerate-devices.exe -c
```

출력 내용:
- **Intrinsic Parameters**: Width/Height, PPX/PPY, Fx/Fy, Distortion, Coeffs, FOV
- **Motion Intrinsic** (IMU 지원 시): Bias/Noise Variances, Sensitivity
- **Extrinsic Parameters**: 스트림 간 회전 행렬·이동 벡터

---

## 자주 쓰는 조합

```powershell
# 장치 요약만
.\rs-enumerate-devices.exe -S

# 캘리브 + 옵션
.\rs-enumerate-devices.exe -c -o

# bag 파일 캘리브 덤프
.\rs-enumerate-devices.exe -c -p record.bag
```

주의: `-s` / `-S`는 `-c`와 **함께 쓸 수 없음** (경고 후 `-c` 무시).

자세한 옵션: `tools/enumerate-devices/readme_kr.md`
