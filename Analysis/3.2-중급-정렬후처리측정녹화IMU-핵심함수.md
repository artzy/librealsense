## rs-depth (C)

**소스:** `examples/C/depth/rs-depth.c`  
**중요 기능:** depth ASCII 히트맵 (콘솔)

| 함수 / API | 설명 |
|------------|------|
| `get_depth_unit_value(dev)` | `rs2_query_sensors()`, `RS2_OPTION_DEPTH_UNITS`로 depth scale 조회 |
| `rs2_get_video_stream_resolution()` | 스트림 해상도 |
| `rs2_get_stream_profile_data()` | stream·format·fps 등 프로파일 정보 |
| ASCII 렌더 루프 | depth 픽셀을 `one_meter` 기준으로 문자 맵에 매핑 (다운샘플 후 출력) |

## rs-post-processing

**소스:** `examples/post-processing/rs-post-processing.cpp`  
**중요 기능:** spatial/temporal 등 필터 비교 (원본 vs 필터 3D)

| 함수 / API | 설명 |
|------------|------|
| `rs2::decimation_filter` | depth 해상도·노이즈 감소 |
| `rs2::rotation_filter` | depth 회전 |
| `rs2::threshold_filter` | 거리 범위 밖 제거 |
| `rs2::disparity_transform` | depth↔disparity 도메인 변환 |
| `rs2::spatial_filter` | 엣지 보존 공간 스무딩 |
| `rs2::temporal_filter` | 시간적 노이즈 감소 |
| `filter_options` | 필터 on/off·옵션 슬라이더 UI |
| `rs2::frame_queue` | 원본·필터 결과 스레드 간 큐 |
| `processing_thread` | `wait_for_frames()` → 필터 체인 → `filtered_data.enqueue()` |
| `original_pc.calculate()`, `filtered_pc.calculate()` | 양쪽 pointcloud 생성 |
| `update_data()` | GL로 원본·필터 pointcloud 동시 표시 |

## rs-motion

**소스:** `examples/motion/rs-motion.cpp`  
**중요 기능:** IMU(gyro+accel)로 카메라 자세 추정

| 함수 / API | 설명 |
|------------|------|
| `check_accel_gyro_are_supported()` | GYRO·ACCEL 스트림 지원 장치 검색 |

| `cfg.enable_stream(RS2_STREAM_GYRO/ACCEL, MOTION_XYZ32F)` | IMU 스트림 설정 |

| `motion.get_motion_data()` | `rs2_vector` (x,y,z) |

| `rotation_estimator::process_accel()` | 가속도→pitch/roll, 상보 필터(alpha) |
| `camera_renderer::render_camera(theta)` | D435 3D 모델 GL 회전 |
| `uncompress_d435_obj()` | 내장 OBJ 압축 데이터 (`d435.h`) |
