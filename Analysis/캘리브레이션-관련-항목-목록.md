## 3. Public API (필수 진입점)

### 헤더

- `include/librealsense2/h/rs_device.h` / `hpp/rs_device.hpp` — OCC·Tare·테이블·triggered
- `include/librealsense2/h/rs_types.h` — `rs2_intrinsics`, `rs2_extrinsics`, extension
- `include/librealsense2/h/rs_sensor.h` — `set/override_intrinsics`, `override_extrinsics`
- `include/librealsense2/h/rs_frame.h` — 타깃 추출, calib 메타데이터
- `include/librealsense2/h/rs_option.h` — `ENABLE_DYNAMIC_CALIBRATION`
- `include/librealsense2/rsutil.h` — project / deproject / FOV

### 주요 함수·클래스

- `run_on_chip_calibration` / `run_tare_calibration`
- `run_focal_length_calibration` / `run_uv_map_calibration` / `calculate_target_z`
- `process_calibration_frame` (host-assist)
- `get/set_calibration_table`, `write_calibration`, `reset_to_factory_calibration`
- `get/set_calibration_config` (D500)
- `trigger_device_calibration` + calibration change callback
- `get_intrinsics` / `get_extrinsics_to` / `get_motion_intrinsics`
- C++: `auto_calibrated_device`, `calibrated_device`, `device_calibration`

## 9. 데이터·에셋

- **Gold/공장 테이블 파일은 리포에 없음** — 장치 EEPROM/Flash에 존재 (`d500_calib_gold` 등은 슬롯 타입)
- OCC/Tare 파라미터는 예제·테스트의 **JSON 문자열**
- IMU 도구 실행 시 `accel_*.txt` / `gyro_*.txt` 생성
- `tools/rs-imu-calibration/images/` — 자세 가이드
- 스티칭용 외부 캘리브 파일 + MATLAB export 스크립트
