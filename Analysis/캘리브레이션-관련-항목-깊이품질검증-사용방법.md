# 캘리브레이션 관련 — 깊이품질검증 사용방법

| 항목 | 내용 |
|------|------|
| 경로 | `tools/depth-quality/` (`rs-depth-quality.exe`) |
| 역할 | 깊이품질검증 |
| 용도 | 캘리브 전후 depth 품질 메트릭 확인 |

---

## 사전 조건

- `build\Release\rs-depth-quality.exe`
- 평면·비반사 타깃 (벽 등)
- 상세: [Depth Quality Testing White Paper](https://dev.realsenseai.com/docs/camera-depth-testing-methodology/)

---

## 실행

```powershell
cd d:\study\librealsense\build\Release
.\rs-depth-quality.exe
```

---

## 측정 절차

1. 카메라를 평면 타깃에서 **0.3 ~ 2 m** 거리에 둔다.
2. 타깃을 조준하고 수 초간 고정 → 3D 뷰에 노란 그리드(평면)가 잡힐 때까지 대기
3. **Angle** 메트릭으로 카메라가 타깃에 수직이 되도록 조정
4. Fill Rate, Plane Fit RMS, Sub-Pixel RMS, Temporal Noise 등 확인
5. Z-Accuracy가 필요하면 UI에서 **Ground Truth**에 실측 거리(mm) 입력
6. **Start_record**로 메트릭·raw 데이터 저장 가능

---

## 주요 메트릭

| 메트릭 | 의미 |
|--------|------|
| Fill Rate | ROI 내 유효(비영) depth 픽셀 비율 |
| Distance to Target | ROI 평균 거리(mm) |
| Plane Fit RMS | 평면 적합 대비 편차 |
| Sub-Pixel RMS | 디스패리티 기반 서브픽셀 오차 |
| Z Accuracy | GT 평면 대비 Z 오차 |
| Temporal Noise | 프레임 간 시간적 노이즈(%) |

캘리브레이션 직후 depth 개선 여부를 이 도구로 검증하는 용도로 쓴다.
