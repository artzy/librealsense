| 항목 | 내용 |
|------|------|
| 경로 | `wrappers/python/examples/d500_triggered_calibration.py` |
| 역할 | D500트리거캘리브 |

## 사전 조건

- Python + `pyrealsense2` + `numpy`
- D500 카메라 연결
- Safety 카메라는 Service Mode에서 캘리브 가능하도록 스크립트가 전환

---

## 실행

```powershell
cd d:\study\librealsense\wrappers\python\examples
python d500_triggered_calibration.py
```

---

## 동작 순서

1. D500 여부 확인 (D585S면 Safety 플래그)
2. Depth 스트리밍 → **fill rate** 측정 (캘리브 전)
3. HWM로 현재 depth calibration 다운로드 → `depth_calib_before.bin` 저장
4. Safety면 **Service Mode** 전환
5. `run_on_chip_calibration("calib run", …)` 트리거
6. 결과 → `depth_calib_after.bin` 저장
7. before/after 테이블이 같으면 실패로 종료
8. 다시 fill rate 측정 → 개선율 비교 출력

---

## 확인 포인트

- 콘솔에 Calibration progress(%) 표시
- fill rate 전후 비교로 캘리브 효과 확인
- 생성 파일: 스크립트 폴더의 `depth_calib_before.bin`, `depth_calib_after.bin`
