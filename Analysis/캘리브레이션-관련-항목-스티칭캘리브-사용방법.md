# 캘리브레이션 관련 — 스티칭캘리브 사용방법

| 항목 | 내용 |
|------|------|
| 경로 | `wrappers/pointcloud/pointcloud-stitching/` |
| 실행 | `rs-pointcloud-stitching` |
| MATLAB | `export_calibratrion.m`, `load_intrinsics.m` |
| 역할 | 스티칭캘리브 |
| 상세 가이드 | `doc/pointcloud-stitching-demo.md` |

---

## 사전 조건

- `BUILD_PC_STITCHING=ON`으로 빌드
- 고정 마운트된 카메라 2대
- (캘리브 단계) MATLAB Stereo Camera Calibrator
- 작업 폴더 (경로에 공백·와일드카드 지양)

---

## 1. IR 이미지 수집용 설정

작업 폴더에 `<시리얼>.cfg` 생성 (`rs-enumerate-devices -s`로 시리얼 확인):

```text
INFRARED,640,480,30,Y8,1
```

두 카메라 모두 Depth에 쓸 해상도와 동일하게 IR을 맞춘다.

```powershell
rs-pointcloud-stitching C:\pc_stitching_ws
```

**Save Frames**로 체커보드를 양쪽 FOV에 보이게 10~20쌍 저장.  
이미지는 `<ws>/calibration_input/` 및 intrinsic 설명 파일이 생성된다.

---

## 2. MATLAB 스테레오 캘리브

1. Stereo Camera Calibrator로 두 카메라 이미지 로드·캘리브
2. `export_calibratrion.m`으로 결과 변환:

```matlab
export_calibratrion(from_serial, to_serial, stereoParams, 'calib.txt')
```

출력: 카메라 간 rotation/translation 및 virtual device용 중간 좌표계 행.

---

## 3. 스티칭 실행

Depth(+Color)용 cfg와 캘리브 파일을 작업 폴더에 두고 다시 실행.  
가상 장치가 생성되면 **realsense-viewer**에서 로드·확인 가능.

주의:
- 실시간 캘리브가 아니라 **사전 계산된 행렬**을 사용한다.
- 카메라 상대 자세가 바뀌면 캘리브를 다시 해야 한다.

전체 단계·마운트 팁: `wrappers/pointcloud/pointcloud-stitching/doc/pointcloud-stitching-demo.md`
