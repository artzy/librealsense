## rs-gl

**소스:** `examples/gl/rs-gl.cpp`  
**중요 기능:** GPU 포인트클라우드·depth colorize (realsense2-gl)

| 함수 / API | 설명 |
|------------|------|
| `rs2::gl::init_processing(app, use_gpu)` | GL 컨텍스트·GPU 처리 초기화 |
| `rs2::gl::init_rendering()` | GL 렌더링 모듈 초기화 |
| `rs2::gl::uploader` | depth를 GPU에 명시적 업로드 |
| `rs2::gl::colorizer` | GPU histogram equalization colorize |
| `rs2::gl::pointcloud` | GPU pointcloud 계산 |
| `upload.process(depth)` | CPU depth → `rs2::gl::gpu_frame` |
| `pc.map_to(depth_texture)`, `pc.calculate(depth)` | GPU texture 매핑·points 생성 |
| `rs2::gl::pointcloud_renderer` | points를 화면에 GL 렌더 |
| `prepare_matrices()` | 마우스 입력 → projection·view 행렬 |
| `handle_input()` | CPU/GPU 처리 토글 (스페이스) |
| `pipe.poll_for_frames(&frames)` | 논블로킹 프레임 (렌더와 분리) |

## rs-software-device

**소스:** `examples/software-device/rs-software-device.cpp`  
**중요 기능:** 가상 장치·합성 프레임·syncer

| 함수 / API | 설명 |
|------------|------|
| `custom_frame_source` | synthetic depth(파동)·color(로고) 픽셀 생성 |
| `rs2::software_device dev` | 소프트웨어 전용 가상 장치 |
| `dev.add_sensor("Depth"/"Color")` | 가상 센서 정의 |
| `sensor.add_video_stream({stream, index, ... intrinsics})` | 가상 비디오 스트림·intrinsics |
| `dev.create_matcher(RS2_MATCHER_DEFAULT)` | 타임스탬프 기반 동기화 |
| `rs2::syncer sync` | 가상 센서 프레임 동기화 |
| `sensor.open()`, `sensor.start(sync)` | 가상 센서 스트리밍 시작 |
| `stream.register_extrinsics_to()` | depth↔color extrinsics 등록 |
| `sensor.on_video_frame({data, deleter, stride, bpp, ts, domain, number, profile, depth_unit})` | 합성 프레임 푸시 |
| `sync.wait_for_frames()` | 동기화된 frameset 수신 |
| `pc.calculate()`, `draw_pointcloud()` | 합성 데이터로 PC 시각화 |
